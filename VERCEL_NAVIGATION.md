# Vercel Navigation - Proje Bulma

## Vercel Ana Sayfası

1. **https://vercel.com** adresine gidin
2. Giriş yaptıktan sonra ana sayfada projeleriniz listelenir
3. **"hadis-arama-motoru"** projesini bulun ve tıklayın

## Proje Sayfasında

Proje sayfasına girdiğinizde şunları görebilirsiniz:

### Üst Kısım
- Proje adı: `hadis-arama-motoru`
- URL (eğer deploy edildiyse): `https://hadis-arama-motoru.vercel.app`
- "Visit" butonu

### Sekmeler
- **Overview** (Genel Bakış)
- **Deployments** (Deploy'lar)
- **Analytics** (İstatistikler)
- **Settings** (Ayarlar)

## Deployments Sekmesi

1. Proje sayfasında **"Deployments"** sekmesine tıklayın
2. Orada tüm deployment'lar listelenir
3. En üstte en son deployment görünür
4. Deployment'a tıklayarak detayları görebilirsiniz

## Eğer Proje Görünmüyorsa

1. **https://vercel.com/new** adresine gidin
2. **"Import Git Repository"** butonuna tıklayın
3. GitHub repository'nizi seçin: `Ahmetvhd/hadis-arama-motoru`
4. **"Import"** butonuna tıklayın
5. Bu yeni bir deployment başlatacak

## URL'yi Bulma

Eğer deployment başarılı olduysa:
- Proje ana sayfasında üstte URL görünür
- Veya Deployment details sayfasında URL görünür
- Genellikle: `https://hadis-arama-motoru.vercel.app`
