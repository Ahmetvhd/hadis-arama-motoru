# GitHub URL Formatı Düzeltme

## ❌ Yanlış Format
```
vercel.com/akoca717-5724s-projects/hadis-arama-motoru
```

## ✅ Doğru Format
```
https://hadis-arama-motoru.vercel.app
```

VEYA

```
https://hadis-arama-motoru-xxxxx.vercel.app
```

## Vercel URL'nizi Bulma

1. **Vercel Dashboard:** https://vercel.com/dashboard
2. Projenizi seçin (`hadis-arama-motoru`)
3. Proje sayfasında üstte **URL** görünecek
4. URL şu formatta olmalı:
   - `https://hadis-arama-motoru.vercel.app`
   - veya `https://hadis-arama-motoru-xxxxx.vercel.app`

## GitHub'a Ekleme

1. Repository sayfanızda sağ üstteki ⚙️ (About düzenle) butonuna tıklayın
2. **"Website"** alanına **TAM URL'yi** ekleyin:
   ```
   https://hadis-arama-motoru.vercel.app
   ```
3. **ÖNEMLİ:** `https://` ile başlamalı!
4. "Save changes" butonuna tıklayın

## Örnek Doğru URL'ler

- ✅ `https://hadis-arama-motoru.vercel.app`
- ✅ `https://hadis-arama-motoru-git-main-ahmetvhd.vercel.app`
- ❌ `vercel.com/akoca717-5724s-projects/hadis-arama-motoru`
- ❌ `hadis-arama-motoru.vercel.app` (https:// olmadan)
